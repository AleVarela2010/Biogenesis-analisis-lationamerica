
## Contexto

Se trata de un proyecto formativo realizado en forma individual, fue mi primer experiencia utiliando python, 
realice una proceso ETL y posteriormente un proceso EDA, para finalmente volcar los insights en un dashboard
de Power BI. El dataset contenia datos de 6 paises de latinoamerica y debia servir para ayudar a tomar una 
decision de expansion de negocio.


## Tecnologias usadas

- Python (con librerias pandas, numpy, matplotlib y seaborn)
- Power BI

## Conclusiones:
  
- Profundizar el analisis a partir de ternas definiendo previamente cual es la prioridad o interes del negocio:

- 1) Volumen de negocio: Brasil y Mexico

- 2) Por volumen de Vacunacion: Brasil y Chile

- 3) Por PBI per Capita: Chile y Argentina

- 4) Por Medicos cada 1000 habitantes: Argentina y Chile

- Recomendaciones: profundizar el analisis incorporando mas informacion comercial, economica, y geopolitica.

- 1) Analisis de la competencia en cada pais

- 2) Analisis de barreras normativas y arancelarias

- 3) Analisis de riesgos de politica economica

- 4) Analisis de transparencia de negocios
